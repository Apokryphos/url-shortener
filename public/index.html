URL Shortener

Redirect

Redirects to the original URL associated with the specified short URL.

URL

GET https://polar-whiskey.glitch.me/<Short URL>

Shorten

Shortens the specified URL.

The specified URL must begin with http:// or https://.

Response properties will be null if the specified URL is invalid.

URL

GET https://polar-whiskey.glitch.me/shorten/?url=<URL>

Example usage:

https://polar-whiskey.glitch.me/shorten/?url=https://www.google.com

Example response:

{

"longUrl": "http%3A%2F%2Fwww.google.com"

"shortUrl": "3",

}
